# prog-101-practice

TypeScript project initialized with basic structure.

## Project Structure

```
prog-101-practice/
├── src/          # TypeScript source code
├── dist/         # Compiled JavaScript
├── css/          # Stylesheets
├── index.html    # Main HTML file
└── README.md
```

## Available Scripts

```bash
npm run build       # Compile TypeScript
npm run dev         # Compile + Browser-Sync (localhost:3000)
npm run start       # Run compiled code
npm run serve       # Only Browser-Sync
npm run lint:css    # Check CSS with Stylelint
npm run lint:css:fix # Fix CSS automatically
```

## Installation

```bash
npm install
```

## Usage

1. Write your TypeScript code in `src/`
2. Compile with `npm run build`
3. Open `index.html` in your browser
